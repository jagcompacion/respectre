# Icon

<!-- STORY -->

### Usage

```js
import { Icon } from 'respectre/elements';
```

### Properties

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   | -            | -          |
| icon      | string   | -            | -          |


### Roadmap

#### Default

```js
import { Icon } from 'respectre/elements';

render() {
  return (
    <Icon icon="icon-forward" />
  );
}
```

#### Sizes

```js
import { Icon } from 'respectre/elements';

render() {
  return (
    <Icon icon="icon-forward" size="2x" />
  );
}
```
